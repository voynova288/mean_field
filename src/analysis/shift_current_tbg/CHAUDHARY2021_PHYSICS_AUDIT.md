# Chaudhary 2021 shift-current physics audit

This note records the current physics interpretation before further numerical tuning.  It is meant to prevent treating parameter sweeps or visual alignment as reproduction.

## 1. Observable and conventions

The plotted quantity is the linearly-polarized shift-current tensor coefficient written in the paper as

\[
J_\mu = 2\,\sigma^{\mu}_{\alpha\alpha}(0,\omega,-\omega) E_\alpha(\omega)E_\alpha(-\omega),
\]

with the main plotted component labelled `sigma_xxy`, equivalent to the code component `y;xx` up to a global sign convention already diagnosed in the paper-comparison plots.

For direct interband transitions, the paper's Eq. (10) is

\[
\sigma^{\mu}_{\alpha\alpha}(0,\omega,-\omega)
= {\pi e^3\over \hbar^2}
\sum_{m,n}\int d^2 k\, f_{mn}\, |A^\alpha_{mn}|^2 S^{\mu\alpha}_{mn}\,\delta(\omega-\epsilon_{mn}).
\]

The code evaluates the equivalent gauge-free expression using `D=<u_n|dH/dk|u_m>`, analytic/finite-difference derivatives, and transition-energy tetra/triangle histograms.  There should be no smoothing; the Lorentzian `eta` is the physical/numerical broadening in the delta function.

A direct quantum-geometry audit has now been run:

```text

```

For component `y;xx` (the current code convention for paper-labelled `xxy`), it checks

\[
R^{xxy}_{mn}=|A^x_{mn}|^2 S^{yx}_{mn}=\mathrm{Im}\left[A^x_{mn}(A^x_{nm})_{;y}\right]
\]

against a gauge-invariant finite-difference link formula for the shift vector.  The max error on deterministic FF/FD audit points is `~4.7e-3 nm`, so the remaining Fig. 2(d,e) map differences should be treated as model/convention discrepancies rather than a broken shift-vector formula.

## 2. Band groups and Pauli logic

Near charge neutrality, band order is

```text
lower dispersive Dv  <  valence flat Fv  <  conduction flat Fc  <  upper dispersive Dc
```

At neutrality, `Dv,Fv` are filled and `Fc,Dc` are empty.

The paper-like same-side FD transitions are

```text
Dv -> Fv      allowed only when holes are present in Fv
Fc -> Dc      allowed only when electrons are present in Fc
```

This explains why FD is zero at neutrality and why `fd_mode=all` was wrong: `all` includes cross-gap `Dv -> Fc` and `Fv -> Dc`, which are active at neutrality and overcount the paper's FD process.

## 3. Why the near-edge FD peak is convention-sensitive

The strongest noninteracting FD peak near `20--22 meV` is a Gamma-region transition.  In the b0 band structure,

```text
Gamma: Dv ~ -41 meV, Fv ~ -20 meV, Fc ~ +19 meV, Dc ~ +40 meV
```

so the same-side FD transition energy is about `20--21 meV`.

For an exact filling-derived chemical potential at `nu=-3.95`, the flat-band count per spin/valley is

\[
N_{\rm flat}/N_k = 1+\nu/4 = 0.0125.
\]

The deepest valence-flat states are precisely near Gamma, so a strictly filling-derived `mu(nu=-3.95)` still leaves the Gamma pocket occupied.  Then `Dv -> Fv` at Gamma is Pauli blocked, and the singular Gamma FD peak is removed.  The analogous statement holds for `nu=+3.95`: a tiny unfilled conduction-flat Gamma pocket blocks the `Fc -> Dc` Gamma peak.

Therefore the paper's very large Fig. 4(b)-style edge peak cannot be reproduced with a literal filling-derived `mu` if the Gamma pocket is retained.  It requires an edge-label convention closer to

```text
nu=-3.95: mu in the lower flat-dispersive gap, below Fv(Gamma)
nu=+3.95: mu in the upper flat-dispersive gap, above Fc(Gamma)
```

Our diagnostic edge chemical potentials (`mu=+-30 meV` noninteracting) implement exactly this and recover the giant Fig. 4(b)-like peak.  This is a physical Pauli-blocking issue, not a plotting issue and not primarily a triangle-quadrature artifact: partial-occupation triangles changed the result only modestly.

## 4. Hartree physics and why one epsilon does not fit all current panels

The Hartree potential is generated by density relative to CNP.  Electron doping builds charge in AA-centered flat-band states and shifts/reshapes the bands so that the electron-side flat band flattens and the hole-side flat band broadens; hole doping does the opposite.  This is the mechanism behind the Fig. 3(b) FF enhancement.

For FD transitions, Hartree does two things at once:

1. It shifts the low-energy same-side gaps near Gamma, producing strong `~25--35 meV` resonances.
2. It changes wavefunctions and transition energies away from Gamma, producing additional opposite-sign structure near `~50--70 meV` in the paper.

The current `T=15 K` diagnostics show the first effect robustly.  However, the relative strength of the low-energy and high-energy FD structures changes strongly with the effective dielectric constant.  The paper's discussion says the dielectric is fitted/renormalized following Ref. 65, not simply the substrate value.

A targeted density-mode audit compared the default two-flat-band source with two full-basis variants:

```text
results/shift_current_tbg/chaudhary2021_hartree_density_mode_audit_lg7_m5_eps10_T15K/density_mode_audit_three_modes.md
```

The `full_delta_occ` mode diagonalizes the full continuum Hamiltonian but subtracts a same-Hamiltonian CNP occupation, so the filled remote sea cancels.  It is nearly identical to `flat`, confirming that carrier leakage out of the central flat pair is not the issue.  The `full_fixed_cnp` mode includes remote-band wavefunction polarization explicitly.  It converges, but weakens the same-side band flattening and does not create the missing strong high-energy/opposite-sign FD feature.  Thus the current evidence does not support replacing the flat-only source with a full-density source as the final model; the more plausible interpretation is still that remote/dispersive bands are folded into the effective screening convention.

The Choi Ref. 65 supplement was then checked directly.  It uses `epsilon=15`, keeps only first-star Hartree Fourier components, and often linearly interpolates potentials from the `nu=±4` endpoints.  These conventions are now audited in:

```text

```

First-star truncation changes the spectra moderately but does not resolve the mismatch; edge-potential linear interpolation reduces FF enhancement and also fails to recover a simultaneous Fig. 3/4 match.

Our scalar Hartree prototype with a single `epsilon_r` and finite-temperature occupations does not yet match all of:

```text

Fig. 4(c) low-energy FD amplitude,
Fig. 4(c) high-energy/opposite-sign FD feature.
```

This mismatch is a physics/convention issue, not solved by a single cosmetic rescale.
